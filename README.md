Task 2-Stock Portfolio Tracker

 📝 Project Description

The **Stock Portfolio Tracker** is a simple Python project that helps users calculate the total value of their stock investments. Users enter stock names and the number of shares they own, while stock prices are stored in a predefined dictionary. The program calculates and displays the total investment value. Optionally, the result can be saved to a **.txt** or **.csv** file.

 🎯 Features

* 📌 Enter stock names and quantities.
* 💹 Uses predefined stock prices.
* 🧮 Calculates total investment value.
* 📊 Displays the final portfolio value.
* 💾 Optionally saves the result to a **.txt** or **.csv** file.

  🛠️ Technologies Used

* 🐍 Python
* 📚 Dictionary
* ⌨️ User Input/Output
* ➕ Basic Arithmetic
* 📂 File Handling (Optional)

📥 Input

* Stock Name
* Quantity of Shares

 📤 Output

* 📌 The user enters the number of stocks.
* 📈 The user provides the stock names and quantities.
* 💰 The program calculates the total investment value using predefined stock prices.
* 🎉 The final portfolio value is displayed as **$1650**.
* ✅ The program executes successfully and finishes without errors.

 📂 Project Files

* 📄 stock_portfolio_tracker.py – Main Python program
* 📄 portfolio.txt *(Optional output file)*
* 📄 portfolio.csv *(Optional output file)*

  🚀 How to Run

1. ▶️ Open the Python file.
2. ⌨️ Enter the stock name and quantity.
3. 📈 View the calculated investment value.
4. 💾 Save the result to a file (optional).
